1. Information Collection


  • The Ticket Bot does not store or retain any user or server data. All interactions, including ticket messages and other data, are processed in real time and remain within Discord's ecosystem.



2. How the Bot Processes Information


  • The Ticket Bot facilitates communication between server members and staff by managing tickets directly within Discord.Any information shared within tickets is visible only to the users involved and server staff with appropriate permissions.


3. Data Security and Privacy


  • Since the Ticket Bot does not store any data, there is no risk of data being accessed, leaked, or shared by the bot itself. All data remains within Discord's platform, governed by Discord's privacy and security policies.


4. Data Sharing


  • The Ticket Bot does not share, sell, or otherwise distribute any user or server data to third parties.


5. User Responsibilities


  • Users should be mindful of the information they share within tickets, as it is visible to server staff with ticket access.It is the responsibility of the server administrators to manage ticket access and ensure privacy for their members.


6. Updates to This Policy


  • We may update this Privacy Policy from time to time to reflect changes in how the bot operates. Changes will be communicated via updates to this document.
