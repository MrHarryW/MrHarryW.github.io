Discord Bot Development

  • Ticket Time

    A multilingual Discord ticket bot available in English, Spanish, French, Arabic, and German, with support for more languages like Russian and Mandarin coming soon. Ticket Time helps server administrators manage user queries efficiently.

  • Watch;Dog

    The steward of "; semi-colon", my developer-focused Discord community. Watch;Dog ensures moderation, upholds order, and facilitates the dissemination of essential information. The bot is open-source, and its source code is available on my website for anyone interested in learning from or contributing to it.

  • Text To Binary

    This Discord bot allows users to seamlessly convert text to binary and vice versa directly within a Discord server. Whether you’re experimenting with binary encoding or just having fun with secret messages, this bot makes it quick and easy!
